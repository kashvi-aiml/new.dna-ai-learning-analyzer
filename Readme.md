# 🧠 AI Learning DNA

AI Learning DNA is an AI-powered student behavioral analytics dashboard designed to analyze learning patterns, productivity consistency, focus stability, wellness balance, and academic engagement using intelligent dataset-driven analysis.

The platform transforms student behavioral indicators into adaptive AI-generated insights, personalized academic recommendations, and productivity-focused learning strategies through a modern interactive dashboard experience.

---

# ✨ Features

- 📊 Student productivity and focus analysis
- 🧠 AI-powered learning insights
- 📈 Behavioral performance scoring
- 🎯 Personalized academic recommendations
- 🗓️ Smart study strategy generation
- 🌌 Modern futuristic AI dashboard UI
- 📂 Dataset-integrated analytics system
- ⚡ Dynamic performance visualization
- ✅ Input validation system
